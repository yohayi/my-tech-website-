# 流式AI环境监控系统 - 官方网站

基于流式处理和实时AI决策的环境智能监控与反馈系统官方网站。

## 功能特点
- 响应式设计，支持手机、平板和电脑
- 完整的技术文档展示
- 案例研究和演示
- 专利信息展示
- 联系表单

## 部署到GitHub Pages
1. 将所有文件上传到GitHub仓库
2. 在仓库设置中启用GitHub Pages
3. 选择main分支作为源
4. 访问 `https://[你的用户名].github.io/[仓库名]`

## 文件结构
- `index.html` - 主页
- `technical.html` - 技术细节
- `demo.html` - 演示案例
- `patent.html` - 专利文档
- `contact.html` - 联系我们
- `css/style.css` - 样式文件
- `js/main.js` - JavaScript文件
- `images/` - 图片资源
- `docs/` - 文档资源

## 自定义
1. 修改 `css/style.css` 中的颜色变量
2. 更新页面内容以匹配您的技术
3. 替换图片资源
4. 添加更多案例或文档
